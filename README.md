# crypt-openssl-keys

OpenSSL key tools.

## Introduction

__Crypt::OpenSSL::Keys__ provides functionality to determine information about
crypto keys. The module is based on [OpenSSL](https://www.openssl.org/).

## Installation

    $ perl Makefile.pl
    $ make install

## Usage

    $ perldoc Crypt::OpenSSL::Keys
